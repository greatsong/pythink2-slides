연산자의 세계

World 4 — 계산의 도구 상자를 열자

pythink2 강의안

연산자는 수학 시간에 배운 것과 비슷하지만, 파이썬만의 특별한 연산자도 있습니다. 특히 //, %, **를 확실히 이해시키는 것이 목표입니다.

오늘 배울 것

  • 파이썬의 나누기 연산자 3총사 (/, //, %)
  • 거듭제곱(**)과 round() 사용법
  • 연산자로 실생활 문제 풀기
기본 사칙연산은 간단히 넘어가고, //, %, **에 집중합니다.

연산자란?

🧰 연산자는 계산의 도구 상자야!
상황에 맞는 도구를 꺼내 쓰면 돼

  • + - * 는 수학이랑 같아
  • / // % 는 나누기의 3가지 버전
  • ** 는 거듭제곱 (수학의 ^)
기본 연산자를 간단히 정리하고, 특수 연산자로 넘어갑니다.

기본 연산자 (복습)


print(10 + 3)    # 13   더하기
print(10 - 3)    # 7    빼기
print(10 * 3)    # 30   곱하기
print(10 / 3)    # 3.333...  나누기 (실수)
/는 항상 소수점이 있는 실수를 돌려줘!
10 / 2도 결과가 5.0이야 (5가 아님!)
/ 연산자는 Python 3에서 항상 float을 반환합니다. 이 점을 미리 짚어두면 // 의 필요성이 자연스럽게 이해됩니다.

// : 몫만 딱!


print(10 // 3)    # 3   (소수점 버림!)
print(17 // 5)    # 3
print(20 // 7)    # 2

🍕 피자 17조각을 5명이 나눠 먹으면?
한 사람당 3조각! (나머지 2조각은 남아)

//는 "몫" → 소수점 아래를 버리고 정수만 남겨!
피자 비유로 몫의 개념을 직관적으로 설명합니다. // 는 floor division이라고 부릅니다.

% : 나머지만 딱!


print(10 % 3)    # 1   (10÷3=3 나머지 1)
print(17 % 5)    # 2   (17÷5=3 나머지 2)
print(20 % 7)    # 6   (20÷7=2 나머지 6)

🍕 피자 17조각을 5명이 나눈 후
남는 2조각이 나머지야!

%는 "나머지" → 나누고 남은 것!
% 연산자를 modulo(모듈로)라고 부릅니다. 짝수/홀수 판별, 시간 계산 등에 유용합니다.

// vs % 한눈에 보기


# 17을 5로 나누면?
print(17 // 5)   # 3  (몫)
print(17 % 5)    # 2  (나머지)

# 검증: 몫 × 나누는수 + 나머지 = 원래수
print(3 * 5 + 2) # 17  맞다!
17 = 5 × 3 + 2
//는 몫    %는 나머지
몫과 나머지의 관계를 수식으로 확인합니다. 이 검증 공식은 문제 풀이에서 자주 사용됩니다.

실전! 시간 변환


total_minutes = 135

hours = total_minutes // 60    # 몫 = 시간
minutes = total_minutes % 60   # 나머지 = 분

print(f"{total_minutes}분 = {hours}시간 {minutes}분")
        

135분 = 2시간 15분
//와 %는 단위 변환에서 정말 자주 쓰여!
분→시간, 초→분, 원→천원 등
시간 변환은 학생들이 공감하기 쉬운 실생활 예제입니다. "영화 135분이면 몇 시간 몇 분?"이라고 물어보세요.

실전! 짝수 / 홀수 판별


num = int(input("숫자: "))

if num % 2 == 0:
    print("짝수!")
else:
    print("홀수!")
어떤 수를 2로 나눈 나머지가
0이면 짝수, 1이면 홀수!
if문은 World 5a에서 자세히 배우지만, 미리 맛보기로 보여줍니다. % 2 == 0 패턴은 매우 자주 사용됩니다.

** : 거듭제곱


print(2 ** 3)     # 8    (2³ = 2×2×2)
print(5 ** 2)     # 25   (5² = 5×5)
print(10 ** 0)    # 1    (어떤 수의 0제곱은 1)
print(9 ** 0.5)   # 3.0  (제곱근! √9 = 3)
수학에서는 ^ 를 쓰지만, 파이썬에서는 **를 써!
**0.5는 제곱근(루트)이야
^ 는 파이썬에서 XOR(비트 연산)이므로 거듭제곱에 쓰면 안 됩니다. 반드시 **를 사용해야 합니다.

round() : 반올림


print(round(3.14159))      # 3    (정수로)
print(round(3.14159, 2))   # 3.14 (소수 2자리)
print(round(3.14159, 1))   # 3.1  (소수 1자리)
print(round(2.5))          # 2    (은행 반올림!)
파이썬의 round()는 은행 반올림(Banker's rounding)!
2.5 → 2, 3.5 → 4 (짝수 쪽으로 반올림)
은행 반올림은 고급 내용이지만, 2.5가 2가 되는 건 학생들이 의아해할 수 있으니 미리 언급합니다. 대부분의 경우에는 우리가 아는 반올림과 같습니다.

연산 순서


# 수학이랑 같아! 곱셈/나눗셈 먼저
print(2 + 3 * 4)      # 14  (3*4=12, +2=14)

# 괄호가 최우선!
print((2 + 3) * 4)    # 20  (2+3=5, *4=20)

# ** > * > + 순서
print(2 ** 3 + 1)     # 9   (2³=8, +1=9)
헷갈리면 괄호를 써!
괄호는 항상 가장 먼저 계산돼
연산 우선순위: ** > *, /, //, % > +, - 헷갈리면 괄호를 쓰라고 안내합니다. 가독성도 좋아집니다.

직접 해보기! (3분)

초(second)를 시:분:초로 변환해 봐!


  1. total_seconds를 입력받기 (예: 3661)
  2. // 와 %로 시간, 분, 초 계산
  3. f-string으로 "1시간 1분 1초" 출력
힌트: 3661초 hours = 3661 // 3600 → 1 남은 = 3661 % 3600 → 61 minutes = 61 // 60 → 1 seconds = 61 % 60 → 1

자주 하는 실수

// 와 % 를 헷갈려! 
# 10을 3으로 나눈 "몫"을 구하고 싶은데...
print(10 % 3)    # 1  ← 이건 나머지야!
암기법: "나누기=// 나머지=%" 
print(10 // 3)   # 3  ← 이게 몫!
print(10 % 3)    # 1  ← 이게 나머지!
외우기: "//"는 두 줄이니까 "몫"(두 글자)
"%"는 동그라미가 있으니까 "나머지"(동전이 남았다!)
//와 %를 헷갈리는 것은 매우 흔한 실수입니다. 재미있는 암기법을 함께 만들어보세요.

자주 하는 실수

0으로 나누기! 
print(10 / 0)    # ZeroDivisionError!
ZeroDivisionError: division by zero
0으로는 절대 나눌 수 없어!
수학에서도 안 되듯이, 파이썬에서도 에러가 나
ZeroDivisionError는 사용자 입력을 받을 때 특히 조심해야 합니다. 나중에 if문으로 방어하는 법을 배울 수 있습니다.

오늘 배운 것 정리

  • /는 일반 나누기 (실수 결과)
  • //는 몫, %는 나머지
  • **는 거듭제곱 (수학의 ^)
  • round(값, 자릿수)로 반올림
  • 헷갈리면 괄호를 써라!
연산자 정리. 특히 //, % 구분을 한 번 더 확인하세요.

꿀팁

"나누기=//, 나머지=%"
이것만 외우면 연산자의 세계 클리어!
단위 변환 = // 와 %
분→시간, 초→분, 센치→미터... 무조건 이 패턴!
짝수 판별 = % 2
n % 2 == 0이면 짝수, 아니면 홀수
핵심 패턴 3가지를 정리합니다. 이 패턴들은 계속 나옵니다.
