World 3 — 사용자의 말을 듣는 귀, input()

pythink2 강의안

이제 일방적으로 출력만 하는 게 아니라, 사용자에게 입력을 받는 방법을 배웁니다. 대화형 프로그램의 시작입니다.

오늘 배울 것

  • input()으로 사용자에게 값 받기
  • input()이 돌려주는 값의 타입 이해하기
  • 형 변환으로 숫자 계산 가능하게 만들기
input()의 핵심은 "항상 문자열을 반환한다"는 것입니다. 이 개념을 확실히 이해시키는 것이 이번 수업의 목표입니다.

input()은 뭘까?

🎤 input()은 마이크야!
사용자가 말하면 그걸 받아 적는 거지

print()가 스피커라면,
input()은 마이크야!

print()=스피커, input()=마이크 비유를 사용합니다. 프로그램이 사용자와 대화하려면 둘 다 필요합니다.

input() 기본 사용법


name = input("이름이 뭐야? ")
print(f"안녕, {name}!")

실행하면:


이름이 뭐야? 민수
안녕, 민수!
input() 안에 쓴 글자가 질문(프롬프트)으로 먼저 나와!
input()의 괄호 안 문자열은 프롬프트입니다. 사용자가 타이핑하고 엔터를 치면 그 값이 변수에 저장됩니다.

input()의 비밀

항상 문자열을 돌려줘!


age = input("나이가 몇 살이야? ")
print(type(age))
print(age)
        

나이가 몇 살이야? 15
<class 'str'>
15
15를 입력했는데... 숫자 15가 아니라 문자열 "15"야!
이것이 이번 수업의 가장 핵심적인 개념입니다. input()은 무조건 문자열(str)을 반환합니다.

문자열 "덧셈"의 함정


a = input("첫 번째 수: ")   # "3" (문자열!)
b = input("두 번째 수: ")   # "5" (문자열!)

print(a + b)   # 뭐가 나올까?
예상: 8
실제: 35 (문자열 "3" + "5" = "35")

문자열끼리 +는 이어붙이기야!

학생들에게 먼저 결과를 예측해보게 하세요. "35"가 나오면 큰 반응이 옵니다. 이 충격을 활용하세요!

해결법: 형 변환!


a = input("첫 번째 수: ")   # "3"
b = input("두 번째 수: ")   # "5"

# 문자열 → 정수로 변환!
a = int(a)
b = int(b)

print(a + b)   # 8 (진짜 덧셈!)

📦➡️🔢 int()는 문자열 상자를 열어서
숫자로 바꿔주는 변환기야!

int()는 integer(정수)의 약자입니다. 문자열을 정수로 변환합니다.

한 줄로 깔끔하게!


# 두 줄로 쓰기
a = input("수 입력: ")
a = int(a)

# 한 줄로 줄이기!
a = int(input("수 입력: "))
int(input(...)) 패턴은 정말 자주 쓰여!
"입력받아서 바로 정수로 바꿔라"는 뜻이야
이 패턴은 외울 정도로 자주 씁니다. 안에서 바깥으로 읽으세요: input()으로 받고 → int()로 변환.

다양한 형 변환


# 정수로 변환
age = int(input("나이: "))         # 15

# 실수로 변환
height = float(input("키: "))     # 165.5

# 문자열로 변환 (잘 안 쓰지만)
text = str(100)                    # "100"
함수변환예시
int()정수로"15" → 15
float()실수로"3.14" → 3.14
str()문자열로100 → "100"
int, float, str 세 가지 형 변환을 정리합니다. float은 소수점이 있는 숫자를 받을 때 사용합니다.

실전 예제: BMI 계산기


height = float(input("키(cm): "))
weight = float(input("몸무게(kg): "))

# cm → m로 변환
height_m = height / 100

# BMI = 몸무게 / 키²
bmi = weight / (height_m ** 2)

print(f"BMI: {round(bmi, 1)}")
        

키(cm): 165
몸무게(kg): 60
BMI: 22.0
실생활 예제로 input()과 형 변환을 복습합니다. float, **, round() 등이 자연스럽게 쓰이는 것을 보여주세요.

이렇게도 할 수 있어!

여러 값 한 번에 입력받기


# split()으로 띄어쓰기 기준 분리
a, b = input("두 수 입력(공백): ").split()
a = int(a)
b = int(b)

print(f"{a} + {b} = {a + b}")
        

두 수 입력(공백): 3 5
3 + 5 = 8
split()은 띄어쓰기 기준으로 문자열을 나눠주는 기능이야
split()은 나중에 문자열 파트에서 자세히 배우지만, 미리 맛보기로 보여줍니다. 이 패턴도 자주 사용됩니다.

더 깔끔한 방법: map()


# map으로 한 번에 int 변환!
a, b = map(int, input("두 수: ").split())

print(f"{a} + {b} = {a + b}")
map(int, ...)은 "다 int로 바꿔!"라는 뜻
여러 숫자를 한 번에 입력받을 때 최고!
map()은 고급 패턴이지만, 알고리즘 문제에서 자주 나오므로 소개합니다. 외우기 어려우면 한 줄씩 변환하는 방법을 써도 괜찮다고 안내하세요.

직접 해보기! (3분)

간단한 계산기를 만들어 봐!


  1. 두 수를 input()으로 입력받기
  2. int()로 변환 후 +, -, *, / 결과 모두 출력
  3. 보너스: 이름도 입력받아서 "OO님의 계산 결과" 형태로!
3분 타이머. 빠른 학생에게는 보너스 과제를 안내하세요.

자주 하는 실수

int() 없이 계산하려고! 
a = input("수: ")    # "10" (문자열)
print(a + 5)          # TypeError!
TypeError: can only concatenate str to str
반드시 int()로 변환 후 계산! 
a = int(input("수: "))
print(a + 5)          # 15
TypeError는 타입이 맞지 않을 때 발생합니다. 문자열과 숫자는 + 연산이 안 된다는 것을 강조하세요.

자주 하는 실수

숫자가 아닌 걸 int()에 넣으면? 
age = int(input("나이: "))
# 사용자가 "열다섯"이라고 입력하면?
ValueError: invalid literal for int()
int()에는 반드시 숫자 형태의 문자열만!
"15" → OK, "열다섯" → 에러, "15.5" → 에러 (float 사용)
ValueError는 값의 형태가 맞지 않을 때 발생합니다. int("15.5")도 에러가 나는데, 이때는 float()를 쓰거나 int(float("15.5"))를 써야 합니다.

오늘 배운 것 정리

  • input()으로 사용자에게 값을 받을 수 있다
  • input()은 항상 문자열을 돌려준다
  • int()로 정수, float()로 실수 변환
  • int(input(...)) 패턴이 가장 많이 쓰인다
  • split()으로 여러 값을 한 번에 입력받을 수 있다
핵심: input()은 항상 문자열! 계산하려면 변환 필수!

꿀팁

int(input()) 외워!
숫자를 입력받을 땐 무조건 이 패턴. 가장 자주 쓰는 코드야
에러 메시지를 읽어봐!

프롬프트에 힌트를 줘!

사용자 친화적인 프롬프트를 작성하는 습관도 중요합니다.
