전략 1: 큰 문제를 잘게 나눠!

Decomposition (분해)
큰 문제 하나를 → 작은 문제 여러 개로!

예: "1~100 중 3의 배수 합 구하기"

  1. 1~100까지 반복 → for + range
  2. 3의 배수인지 확인 → if + %
  3. 합 누적 → 변수 + 누적 패턴
  4. 결과 출력 → print + f-string
분해(Decomposition)는 CT의 4대 요소 중 하나입니다. 문제를 한 번에 풀려 하지 말고, 단계별로 나누는 습관을 들이세요.

분해 → 코드로!


# 1~100 중 3의 배수 합 구하기

total = 0                    # 3. 합 누적 변수

for i in range(1, 101):      # 1. 1~100 반복
    if i % 3 == 0:           # 2. 3의 배수 확인
        total += i           # 3. 누적

print(f"3의 배수 합: {total}")  # 4. 결과 출력
        

3의 배수 합: 1683
각 줄이 분해한 단계 하나에 대응해!
분해만 잘 하면 코드는 자연스럽게 나와
분해한 각 단계가 코드의 각 줄에 대응하는 것을 보여주세요. 주석으로 단계 번호를 표시하면 더 명확합니다.

전략 2: 변수를 추적해!

Variable Tracking (변수 추적)
각 단계에서 변수 값이 어떻게 변하는지 표로 따라가기 

x = 10
y = 3
while x > 0:
    x = x - y
    print(f"x={x}")
회차x (시작)x - yx (끝)
11010-37
277-34
344-31
411-3-2 (종료!)
변수 추적표는 코드를 이해하고 디버깅하는 가장 확실한 방법입니다. 학생들에게 종이에 직접 그려보게 하세요.

종합 예제: 자릿수 분리

"세 자리 수의 각 자릿수를 분리해서 합 구하기"


num = int(input("세 자리 수: "))

hundreds = num // 100
tens = (num // 10) % 10
ones = num % 10

total = hundreds + tens + ones
print(f"{hundreds} + {tens} + {ones} = {total}")
        

세 자리 수: 357
3 + 5 + 7 = 15
// 와 %의 실전 활용입니다. 357을 예로: 357//100=3, (357//10)%10=5, 357%10=7

이렇게도 할 수 있어!

문자열로 자릿수 분리


num = input("세 자리 수: ")   # 문자열로 받기!

total = int(num[0]) + int(num[1]) + int(num[2])
print(f"자릿수 합: {total}")

반복문으로!


num = input("숫자: ")
total = 0
for digit in num:
    total += int(digit)
print(f"자릿수 합: {total}")
같은 문제를 여러 방법으로 풀 수 있어!
하나의 문제, 다양한 사고의 경로
pythink2의 철학 "하나의 문제, 다양한 사고의 경로"를 직접 체험합니다. //% 방식, 문자열 인덱싱, 반복문 — 세 가지 접근법이 모두 유효합니다.

종합 예제: 소수 판별

"입력받은 수가 소수인지 판별하기"

분해:

  1. 수 입력받기 (input + int)
  2. 2부터 n-1까지 반복 (for + range)
  3. 나누어 떨어지면 소수 아님 (if + %)
  4. 결과 출력 (print)

n = int(input("수: "))
is_prime = True

for i in range(2, n):
    if n % i == 0:
        is_prime = False
        break

if n < 2:
    is_prime = False

if is_prime:
    print(f"{n}은(는) 소수!")
else:
    print(f"{n}은(는) 소수가 아님")
소수 판별은 여러 개념이 종합된 대표적 문제입니다. for, if, %, break, 플래그 변수(is_prime)가 모두 사용됩니다.

종합 예제: 양수/음수 개수 세기


n = int(input("몇 개의 수? "))
pos = 0     # 양수 개수
neg = 0     # 음수 개수
zero = 0    # 0 개수

for i in range(n):
    num = int(input(f"{i+1}번째 수: "))
    if num > 0:
        pos += 1
    elif num < 0:
        neg += 1
    else:
        zero += 1

print(f"양수: {pos}, 음수: {neg}, 0: {zero}")
카운팅 패턴 + 조건 분기 조합!
for + if-elif-else + 카운팅 패턴의 조합입니다. 변수 3개를 동시에 추적하는 연습이 됩니다.

종합 예제: 모음 세기


text = input("영어 문장: ").lower()
vowels = "aeiou"
count = 0

for char in text:
    if char in vowels:
        count += 1

print(f"모음 개수: {count}")
        

영어 문장: Hello Python
모음 개수: 3
문자열 순회(for) + 포함 검사(in) + 카운팅 = 텍스트 분석!
.lower()로 대소문자를 통일한 후 처리하는 패턴입니다. World 5b의 문자열 + World 5a의 조건 + World 6의 반복이 합쳐진 예제입니다.

디버깅: 코드가 안 될 때

  1. print()로 찍어봐!
    print(f"DEBUG: x={x}")
  2. 변수 추적표를 그려!
    종이에 회차별로 변수 값 적기
  3. 경계값을 넣어봐!
    0, 1, 최대값, 음수 등
  4. 한 줄씩 읽어!
    코드를 천천히 소리 내어 읽기
디버깅은 프로그래밍의 핵심 스킬입니다. "버그가 있는 코드"가 아니라 "아직 완성되지 않은 코드"라고 생각하게 하세요.

종합 문제: 비밀번호 검증기


  • 비밀번호를 입력받기
  • 길이가 8자 이상인지 확인
  • 숫자가 하나 이상 포함되어 있는지 확인
  • 조건을 만족하면 "강한 비밀번호!"
    아니면 어떤 조건이 부족한지 알려주기
5분으로 시간을 좀 더 줍니다. 종합 문제이므로 시간이 더 필요합니다. 힌트: len()으로 길이 확인, for + isdigit()로 숫자 포함 확인

자주 하는 실수

elif를 빼먹고 if만 나열! 
score = 85
if score >= 90:
    grade = "A"
if score >= 80:     # elif여야 해!
    grade = "B"
if score >= 70:     # elif여야 해!
    grade = "C"
# 결과: grade = "C" (B가 아님!)
연속 조건은 elif로!
if-if-if는 모두 독립적으로 검사해!
if-elif-elif는 첫 번째 True에서 멈춰!
if-if-if와 if-elif-elif의 차이는 매우 중요합니다. if-if-if는 모든 조건을 독립적으로 검사하므로, 마지막 조건이 덮어씁니다.

자주 하는 실수

음수 처리를 잊어! 
# 두 수의 차이를 구하고 싶은데...
a = 3
b = 7
diff = a - b
print(diff)    # -4  (음수가 나와!)
abs()로 절댓값! 
diff = abs(a - b)
print(diff)    # 4
abs() = absolute(절대적인)의 약자
"차이"를 구할 때는 abs()를 꼭 기억해!
abs()는 간단하지만 자주 빠뜨리는 함수입니다. "두 수의 차이"를 구할 때 항상 abs()를 생각하게 하세요.

문제 풀이 5단계

  1. 1 이해: 문제가 뭘 요구해?
  2. 2 분해: 작은 단계로 나누기
  3. 3 연결: 어떤 "무기"를 쓸지 결정
  4. 4 코딩: 한 단계씩 코드로
  5. 5 검증: 경계값으로 테스트
이 5단계 프로세스를 습관화하면 어떤 문제든 접근할 수 있습니다. 특히 2단계(분해)가 가장 중요합니다.

무기 총정리!

  • print() + f-string: 출력의 기본
  • 변수 + =: 데이터 저장과 계산
  • int(input()): 입력받아서 숫자로
  • // % **: 몫, 나머지, 거듭제곱
  • if elif else: 조건에 따라 분기
  • [인덱싱] [슬라이싱]: 문자열 다루기
  • for while: 반복의 힘
전체 무기를 총정리합니다. 이 7가지만 잘 조합하면 대부분의 기초 문제를 풀 수 있습니다.
