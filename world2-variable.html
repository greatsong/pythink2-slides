World 2 — 데이터를 담는 이름표 상자

pythink2 강의안

변수는 프로그래밍의 가장 기본적인 개념입니다. 상자 비유를 통해 직관적으로 이해시켜 주세요.

오늘 배울 것

  • 변수에 값을 저장하고 꺼내 쓰기
  • 변수로 계산하기
  • 두 변수의 값을 교환(swap)하기
세 가지 학습 목표를 하나씩 보여주세요.

변수란 뭘까?

📦 변수는 이름표가 붙은 상자야!

  • 상자에 값을 넣고 (저장)
  • 이름표를 보고 꺼내 쓰고 (사용)
  • 새 값으로 바꿀 수도 있어 (재할당)
상자 비유를 손으로 그리며 설명해도 좋습니다. "name"이라고 적힌 상자에 "철수"를 넣는 그림을 상상해보세요.

변수 만들기


# 이름표 = 값
name = "파이썬"
age = 15
height = 165.5
is_student = True

=는 "같다"가 아니라 "넣는다"는 뜻!

수학에서 = 은 "같다"지만,
파이썬에서 = 은 "오른쪽 값을 왼쪽 상자에 넣어라"
= 기호의 의미 차이를 강조합니다. 이건 정말 중요한 개념입니다. "같다"는 ==으로 나중에 배울 거라고 미리 언급해도 좋습니다.

변수 이름 규칙

이렇게 지어! 
student_name = "영희"
score1 = 95
my_age = 15
이건 안 돼! 
1score = 95       # 숫자로 시작
my name = "영희"  # 띄어쓰기
for = 10          # 예약어
규칙: 영문/숫자/밑줄(_)만 사용, 숫자로 시작 불가, 의미 있는 이름!
변수 이름은 의미 있게 짓는 습관이 중요합니다. a, b, x 같은 이름보다 score, name 같은 이름이 좋다고 강조하세요.

변수 사용하기


name = "민수"
age = 14
school = "중앙중학교"

print(f"이름: {name}")
print(f"나이: {age}살")
print(f"학교: {school}")
        

이름: 민수
나이: 14살
학교: 중앙중학교
World 1에서 배운 f-string을 변수와 함께 사용합니다. 변수에 저장해두면 여러 번 사용할 수 있다는 점을 강조하세요.

변수 값 바꾸기


score = 80
print(f"현재 점수: {score}")

score = 95
print(f"바뀐 점수: {score}")
        

현재 점수: 80
바뀐 점수: 95

같은 상자에 새 값을 넣으면 이전 값은 사라져!

변수의 값은 덮어씌워진다는 걸 강조합니다. 상자에 새 물건을 넣으면 이전 물건은 사라지는 것과 같습니다.

변수로 계산하기


price = 1500       # 아이스크림 가격
count = 3          # 개수

total = price * count
print(f"총 가격: {total}원")
        

총 가격: 4500원

변수끼리 계산한 결과를 새 변수에 저장할 수 있어!

변수의 진짜 힘은 계산에 있습니다. 가격이 바뀌면 price만 바꾸면 total도 자동으로 바뀐다고 설명하세요.

변수 값 누적하기


money = 10000      # 용돈

money = money - 3000   # 점심 3000원
print(f"점심 후: {money}원")

money = money - 2000   # 간식 2000원
print(f"간식 후: {money}원")
        

점심 후: 7000원
간식 후: 5000원
money = money - 3000은
"money 상자에서 꺼내서 3000 빼고 다시 넣어"라는 뜻!
money = money - 3000은 학생들이 처음에 헷갈려합니다. = 이 "같다"가 아니라 "넣는다"라는 걸 다시 한번 강조하세요.

두 변수 값 교환하기 (Swap)

a에는 10, b에는 20이 들어있어.
두 값을 바꾸고 싶어!

이러면 안 돼! 
a = 10
b = 20
a = b    # a = 20
b = a    # b = 20 ???  (10이 사라졌어!)
swap 문제를 먼저 "잘못된 방법"으로 보여주세요. 왜 안 되는지 학생들에게 물어보세요.

Swap: 임시 상자를 써!


a = 10
b = 20

temp = a     # 임시 상자에 a 보관
a = b        # a에 b 값 넣기
b = temp     # b에 임시 상자 값 넣기

print(f"a = {a}, b = {b}")
        

a = 20, b = 10

🥤 컵 두 개의 음료를 바꾸려면?
빈 컵(temp)이 하나 더 필요해!

음료수 컵 비유가 효과적입니다. 콜라와 사이다를 바꾸려면 빈 컵이 필요하듯이요.

이렇게도 할 수 있어!

파이썬만의 특별한 swap


a = 10
b = 20

a, b = b, a   # 파이썬 마법!

print(f"a = {a}, b = {b}")
        

a = 20, b = 10
파이썬은 a, b = b, a 한 줄로 swap이 가능해!
다른 언어에는 없는 파이썬만의 멋진 기능이야
파이썬의 동시 할당(simultaneous assignment)입니다. 다만 temp를 쓰는 방법도 이해해야 한다고 강조하세요 - 다른 언어에서는 필수이고, 사고력 문제에서도 자주 나옵니다.

여러 변수 한 번에 만들기


# 한 줄에 여러 변수!
x, y, z = 1, 2, 3
print(f"x={x}, y={y}, z={z}")

# 같은 값으로 여러 변수
a = b = c = 0
print(f"a={a}, b={b}, c={c}")
        

x=1, y=2, z=3
a=0, b=0, c=0
다중 할당은 코드를 간결하게 만들어줍니다. 하지만 가독성을 위해 남용하지 않는 것이 좋다고 안내하세요.

자주 하는 실수

정의 안 한 변수 사용 
print(score)  # score를 만든 적이 없어!
NameError: name 'score' is not defined
대소문자가 달라! 
Name = "철수"
print(name)    # Name ≠ name
NameError: name 'name' is not defined
변수 이름은 대소문자를 구분해!
Name, name, NAME 은 모두 다른 변수야
NameError는 변수를 만들기 전에 사용하거나, 이름을 잘못 쳤을 때 발생합니다. 대소문자 구분은 초보자가 자주 실수하는 부분입니다.

오늘 배운 것 정리

  • 변수 = 값으로 데이터를 저장한다
  • =은 "같다"가 아니라 "넣는다"
  • 변수끼리 계산해서 새 변수에 저장할 수 있다
  • swap은 temp 또는 a, b = b, a
  • 변수 이름은 의미 있게, 대소문자 구분!
체크리스트를 하나씩 확인하며 복습합니다.
