  • 따옴표의 종류와 사용법 알기

print()는 뭘까?

🗣️ print()는 파이썬이 말하는 방법이야!

우리가 말을 하려면 입을 열듯,
파이썬이 뭔가를 보여주려면 print()를 써야 해

비유를 통해 print()의 역할을 직관적으로 설명합니다. "파이썬은 print()가 없으면 벙어리야"라고 표현해도 좋습니다.

첫 번째 코드!


print("안녕, 파이썬!")

실행 결과:


안녕, 파이썬!
학생들에게 pythink2에서 직접 타이핑하게 하세요. 복사-붙여넣기가 아니라 직접 치는 것이 중요합니다.

따옴표의 종류


# 작은따옴표
print('안녕하세요')

# 큰따옴표
print("안녕하세요")

# 둘 다 똑같이 동작해!
작은따옴표든 큰따옴표든 결과는 같아! 편한 걸 골라 써
작은따옴표와 큰따옴표의 차이가 없다는 것을 강조합니다. 다만 일관성 있게 사용하는 습관을 권장하세요.

따옴표 안에 따옴표?


# 큰따옴표 안에 작은따옴표
print("파이썬이 '안녕'이라고 했어")

# 작은따옴표 안에 큰따옴표
print('그는 "멋져"라고 말했다')

바깥과 안쪽 따옴표를 다르게 쓰면 돼!

따옴표 중첩은 학생들이 자주 헷갈리는 부분입니다. 규칙은 간단합니다: 바깥과 안쪽을 다르게!

줄 바꾸기: \n


print("첫째 줄\n둘째 줄\n셋째 줄")

실행 결과:


첫째 줄
둘째 줄
셋째 줄

\n은 엔터 키를 코드로 쓴 거야!

\n은 "new line"의 약자입니다. 역슬래시+n이 합쳐져서 줄바꿈 역할을 합니다.

이렇게도 할 수 있어!

삼중 따옴표 (Triple Quotes)


print("""첫째 줄
둘째 줄
셋째 줄""")
여러 줄을 출력할 때는 삼중 따옴표가 더 편해!
\n을 안 써도 되거든
삼중 따옴표는 \n보다 직관적입니다. 긴 텍스트 출력에 유용합니다. """ 또는 ''' 둘 다 가능합니다.

여러 값 한번에 출력하기


print("이름:", "파이썬")
print("나이:", 30, "살")
print(1, 2, 3, 4, 5)

실행 결과:


이름: 파이썬
나이: 30 살
1 2 3 4 5

쉼표로 구분하면 자동으로 띄어쓰기가 들어가!

print()의 인자를 쉼표로 구분하면 자동으로 공백이 삽입됩니다. sep 파라미터를 나중에 다룰 수도 있습니다.

f-string: 변수를 문장에 넣기


name = "철수"
age = 15

print(f"안녕, 나는 {name}이야!")
print(f"나이는 {age}살이야")

실행 결과:


안녕, 나는 철수이야!
나이는 15살이야
f-string은 Python 3.6+에서 사용 가능합니다. 따옴표 앞에 f를 붙이고, 변수를 중괄호 {}로 감쌉니다.

f-string 안에서 계산도 돼!


price = 3000
count = 4

print(f"총 가격: {price * count}원")
print(f"한 개당: {price}원 x {count}개")

실행 결과:


총 가격: 12000원
한 개당: 3000원 x 4개
중괄호 {} 안에는 변수뿐 아니라 계산식도 넣을 수 있어!
f-string의 강력함을 보여주는 예제입니다. 중괄호 안에 어떤 표현식이든 넣을 수 있습니다.

print()의 숨겨진 기능: end


# 기본: 줄바꿈
print("가")
print("나")

# end로 줄바꿈 대신 다른 걸 넣기
print("가", end=" ")
print("나")

기본

가
나

end=" "

가 나
end 파라미터는 print()가 끝날 때 뭘 붙일지 정합니다. 기본은 "\n"(줄바꿈)이지만, 원하는 걸로 바꿀 수 있습니다.

자주 하는 실수

괄호를 안 닫았어! 
print("안녕"
SyntaxError: unexpected EOF
괄호를 꼭 닫아주자 
print("안녕")
가장 흔한 실수입니다. 괄호와 따옴표는 항상 짝이 맞아야 합니다. 에디터의 자동 완성 기능을 활용하라고 안내해주세요.

자주 하는 실수

따옴표 짝이 안 맞아! 
print("안녕')
SyntaxError: EOL while scanning string
시작과 끝 따옴표를 같게! 
print("안녕")
f를 깜빡했어! 
name = "철수"
print("안녕, {name}아")

출력: 안녕, {name}아 (변수가 안 들어감!)

f-string에서 f를 빼먹는 실수가 매우 흔합니다. 에러가 나지 않고 그냥 {name}이 그대로 출력되기 때문에 더 헷갈립니다.

오늘 배운 것 정리

  • print()로 화면에 출력할 수 있다
  • 따옴표는 작은따옴표, 큰따옴표 둘 다 OK
  • \n으로 줄바꿈, 삼중 따옴표로 여러 줄
  • f"..."로 변수를 문장에 넣을 수 있다
  • end=" "로 줄바꿈 대신 다른 문자 넣기
